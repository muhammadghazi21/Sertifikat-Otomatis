Berikut adalah langkah-langkah penggunaan aplikasi ini, simak setiap instruksi yang kami berikan dengan seksama.

Set Up Email

Untuk menggunakan aplikasi ini, Anda harus memiliki akun email yang akan digunakan sebagai pengirim email. Jika Anda belum memiliki akun email, silahkan buat akun email terlebih dahulu atau memakai akun email yang sudah ada. Setelah memiliki akun email, Anda harus buka pada manajemen akun email seperti gambar berikut.

Setelah itu, Anda harus mengaktifkan akses aplikasi pihak ketiga dengan masuk menu "Security" seperti gambar berikut.

Cari sub menu bernama "Sign in to Google" dan pencet menu "App Passwords" seperti gambar berikut.

Pada menu "App Passwords", Anda harus memilih "Select the app and device you want to generate the app password for" dan memilih "Other (Custom name)" pada menu "Select app" dan "Computer" pada menu "Select device". Kemudian masukkan nama aplikasi yang Anda inginkan (Saya memberinya nama "PythonApp") pada menu "Enter the custom name" dan pencet tombol "Generate".

Setelah itu, akan muncul password yang akan digunakan untuk mengakses akun email Anda. Password tersebut akan digunakan untuk pengiriman email pada aplikasi ini. Jangan lupa untuk menyimpan password tersebut karena akan digunakan untuk mengakses akun email Anda.

Set up akun email Anda selesai. Berikutnya kita set up data peserta pada excel.

Set Up Data pada excel

Untuk menggunakan aplikasi ini, Anda harus memiliki data peserta dalam bentuk excel. Data peserta harus berisi nama peserta, peran peserta, dan email peserta. Data peserta harus berada pada sheet pertama pada excel dan dengan penamaan yang berurutan. Berikut adalah contoh data peserta yang benar.

Set Up Template

Setelah itu, Anda harus membuat template sertifikat yang akan digunakan pada setiap sertifikat yang dicetak. Template sertifikat harus berupa file word dengan format .docx. Walaupun begitu template dapat berupa gambar didalam .doc akan tetapi komponen utama yang harus ada adalah format tempat pengisian nama dan peran peserta yang berbentuk "{{ Nama }}" dan "{{ Sebagai }}". Berikut adalah contoh template sertifikat yang benar.

Pastikan kedua file diatas tidak dalam keadaan terkunci atau protected sehingga program tidak mampu mengolahnya.

Setelah Tiga tahapan diatas dilakukan, Anda sudah dapat menggunakan aplikasi ini. Untuk menggunakan aplikasi ini, Anda harus menjalankan file "generate-sertifikat.py" pada folder "src". Setelah itu, Anda akan diarahkan untuk melalui menu dan pengisian semua komponen yang anda miliki sendiri (.xlsx, .docx, dan password email).
